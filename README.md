# Vite + React + TypeScript + UnoCSS + shadcn/ui 模板

一个轻量模板，集成：

- Vite + React + TypeScript
- UnoCSS（含 attributify / icons / typography preset）
- shadcn/ui 风格组件基础（示例 `Button`）
- `@/*` 路径别名

## 快速开始

```bash
npm install
npm run dev
```

## 可用脚本

- `npm run dev`：本地开发
- `npm run build`：类型检查 + 生产构建
- `npm run preview`：预览构建结果
- `npm run typecheck`：TypeScript 类型检查

## 目录结构

```txt
src/
  components/ui/button.tsx
  lib/utils.ts
  App.tsx
  main.tsx
  styles.css
```
